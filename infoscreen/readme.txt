myWebLog INFORMATION SCREEN v2 INSTRUCTIONS
===========================================

1. Place all files (infoscreen.html, infoscreen_v2.css and gotoplane.gif) in a folder on your webserver or local computer.

2. Open infoscreen.html in Notepad or similar plain text editor. Insert you club ID and authorization code on line 8. Also remove [ and ].
   Your autorization code can be found under "Administration|Important admin info" when you have logged in.
   
3. Open infoscreen.html in a web browser and activate the full screen function. Exactly how this is done depends on the browser, but F11 usually works on PCs.

4. DONE!
